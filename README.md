🏢 Sunflag Nig Ltd Visitor Management System

A modern, mobile-friendly visitor registration system built for Sunflag Nig Ltd.
This application allows visitors to register via QR code, while securely storing and managing entry/exit records.

⸻

🚀 Features
 • 📱 QR Code Registration – Visitors scan and fill the form on their device
 • ⏱ Automatic Time Tracking – Real-time Time In and Time Out
 • ✅ Digital Acknowledgement – Visitors confirm submitted details
 • 📊 Admin Dashboard – View and manage all visitor records
 • 📥 Daily Excel Reports – Export visitor logs instantly
 • 🎨 Modern UI – Mobile-friendly with clean design and animations
 • 🔄 Live Feedback – Success message after submission

⸻

🧱 Tech Stack
 • Backend: Python (Flask)
 • Database: SQLite
 • Frontend: HTML, CSS, Bootstrap
 • Other Tools: QR Code Generator, OpenPyXL
